# App Switcher

Visual app switcher for Raycast that shows running macOS apps and their windows in a fast, keyboard-friendly interface.

## Commands

- `Switch Apps`: Browse running apps and windows, then switch, close, minimize, hide, or quit.
- `Manage Excluded Apps`: Review and remove apps excluded from the switcher.

## Preferences

- `App Ordering`: Most Recently Used or Alphabetical.
- `Icon Size`: Small, Medium, or Large.
- `View Mode`: Grid or List.

## Notes

- This extension is macOS-only and relies on AppleScript/JXA window automation APIs.
